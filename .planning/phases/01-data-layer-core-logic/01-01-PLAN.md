    Create internal/task/task.go with the following:

    1. **Imports**: crypto/rand, encoding/hex, time

    2. **generateUUID function** (per D-01):
       ```go
       func generateUUID() (string, error) {
           b := make([]byte, 16)
           if _, err := rand.Read(b); err != nil {
               return "", err
           }
           return hex.EncodeToString(b), nil // 32-char hex string
       }
       ```

    3. **Task struct** with JSON tags:
       ```go
       type Task struct {
           ID        string    `json:"id"`
           Title     string    `json:"title"`
           Category  string    `json:"category,omitempty"`
           Completed bool      `json:"completed"`
           CreatedAt time.Time `json:"created_at"`
       }
       ```

    4. **NewTask constructor**:
       ```go
       func NewTask(title, category string) (*Task, error) {
           id, err := generateUUID()
           if err != nil {
               return nil, err
           }
           return &Task{
               ID:        id,
               Title:     title,
               Category:  category,
               Completed: false,
               CreatedAt: time.Now(),
           }, nil
       }
       ```

    Note: Do NOT add UpdatedAt field (per research, Phase 1 doesn't need it).

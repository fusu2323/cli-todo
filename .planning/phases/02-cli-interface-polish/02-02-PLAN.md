  <action>
    Create cmd/main.go with the following structure:

    1. Imports: errors, flag, fmt, os, github.com/fusu2323/cli-todo/internal/task

    2. main() function:
       - Check len(os.Args) < 2: print global help and exit 0
       - Create store via task.NewJSONFileStore("")
       - Switch on os.Args[1]: add, list, done, delete, help, default (unknown subcommand)
       - For subcommand cases, call handleXxx(store) with os.Args passed
       - For unknown subcommand: print "unknown subcommand: <name>" to stderr, print help, exit 1

    3. handleAdd(store):
       - Create FlagSet: flag.NewFlagSet("add", flag.ExitOnError)
       - Define category flag: fs.String("c", "", "category")
       - Set fs.Usage to print "Usage: todo add <title> [-c category]\n\nFlags:"
       - Parse os.Args[2:] (NOT os.Args[1:])
       - Check fs.NArg() < 1: call fs.Usage() and exit 1
       - Call task.NewTask(fs.Arg(0), *category)
       - Call store.Add(t)
       - On any error: fmt.Fprintln(os.Stderr, "error:", err) and os.Exit(1)
       - Silent success (no output on success per D-01)

    4. handleList(store):
       - Create FlagSet for "list"
       - Define category flag: fs.String("c", "", "category")
       - Set fs.Usage to print "Usage: todo list [-c category]\n\nFlags:"
       - Parse os.Args[2:]
       - Call store.List(*category)
       - For each task, print "[x] Title @category" format (D-03)
         - If task.Completed: check = "x", else check = " "
         - If task.Category != "": fmt.Printf("[%s] %s @%s\n", check, t.Title, t.Category)
         - Else: fmt.Printf("[%s] %s\n", check, t.Title)
       - On error: stderr output and exit 1

    5. handleDone(store):
       - Create FlagSet for "done"
       - Set fs.Usage to print "Usage: todo done <id>\n\nFlags:"
       - Parse os.Args[2:]
       - Check fs.NArg() < 1: fs.Usage() and exit 1
       - Call store.MarkDone(fs.Arg(0))
       - Check errors.Is(err, task.ErrTaskNotFound): print "task not found" to stderr, exit 1
       - On other error: print "error:" + err to stderr, exit 1
       - Silent success

    6. handleDelete(store):
       - Create FlagSet for "delete"
       - Set fs.Usage to print "Usage: todo delete <id>\n\nFlags:"
       - Parse os.Args[2:]
       - Check fs.NArg() < 1: fs.Usage() and exit 1
       - Call store.Delete(fs.Arg(0))
       - Check errors.Is(err, task.ErrTaskNotFound): print "task not found" to stderr, exit 1
       - On other error: print "error:" + err to stderr, exit 1
       - Silent success

    7. printGlobalHelp():
       - Print "Usage: todo <command> [arguments]\n\nCommands:\n  add <title> [-c category]    Add a new task\n  list [-c category]           List all tasks\n  done <id>                     Mark a task as complete\n  delete <id>                   Delete a task\n  help                          Show this help message"
